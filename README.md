- [06 - Multifile Compilation](https://github.com/franciuto/C4School/tree/main/06%20-%20Compilazione%20Multifile) - Multi-file compilation example with assembly generation
- [07 - Prime Numbers](https://github.com/franciuto/C4School/tree/main/07%20-%20Numeri%20primi) - Comparison between Sieve of Eratosthenes and classic algorithm
- [08 - Vector Calculator](https://github.com/franciuto/C4School/tree/main/08%20-%20Calcolatrice%20di%20vettori) - Vector operations library

- [11 - MyNetLib](https://github.com/franciuto/C4School/tree/main/11%20-%20MyNetLib) - IP address operations library (binary conversion, validation, network address calculation)
- [12 - Memory Layout](https://github.com/franciuto/C4School/tree/main/12%20-%20Memory%20layout) - Memory allocation analysis across sections (.text, .data, .bss, stack, heap)
- [13 - IP Generator](https://github.com/franciuto/C4School/tree/main/13%20-%20ip_generator) - Random IP address generator with class and type classification
